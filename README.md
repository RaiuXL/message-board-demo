# Staff Message Board

- **Stack:** Next.js 16 (app router + Turbopack) frontend, Node/Express + Sequelize backend, MySQL + SQLite for tests.
- **Docs:** see `docs/message-board-spec.md` and `docs/assignment-automated-testing.md` for full requirements.
- **Setup:** `cd api && npm install`, `cd frontend && npm install`.
- **Unit tests:** `cd api && npm test`, `cd frontend && npm test`.
- **Playwright suites:** `npm run integration:test` and `npm run e2e:test` from `frontend/` (scripts spawn both servers automatically).
